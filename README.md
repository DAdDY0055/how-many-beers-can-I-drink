## デプロイ先
https://how-many-beers-can-i-drink.herokuapp.com/

## 使い方
- UPのボタンを押すことで飲んだビールの量を増やせます
- DOWを押すと飲んだビールの量を増やせます

(注意)
- 飲んだビールは上から増えます
- カウントは無限にされるので、押しすぎるととんでもないことになります
